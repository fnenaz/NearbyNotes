Application Description
This application is a simple decentralized note system built using Soroban smart contracts. It is designed to store and manage data of nearby users from the LINE Nearby feature. Each stored data represents a record containing basic information such as name, LINE ID, distance, and a beauty rating.
The application focuses on simplicity, efficiency, and minimal resource usage, making it suitable for blockchain-based environments.

Features


Create Data
Allows users to add new records with information: name, LINE ID, distance, and beauty rating (scale 1–10).


Read Data
Enables users to retrieve and view all stored records from the smart contract.


Delete Data
Allows users to delete specific records based on the LINE ID.


On-chain Storage
All data is stored securely within the blockchain using Soroban storage.


Lightweight System
Designed with minimal complexity to ensure efficient execution in a smart contract environment.
